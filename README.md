Proyecto para el Sprint 5 - Herramientas de Desarrollo de Software

1) Histograma
2) Gráfico de Dispersión

Estos gráficos contienen información de:

**Historgrama**
Muestra la cantidad de coches en función de los rangos de kilometraje.

**Dispersión**
